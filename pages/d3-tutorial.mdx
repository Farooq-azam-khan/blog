# 2. Scalable Vector Graphics (SVG)
Use the `<svg>` tag to make Scalable Vector Graphs. 
- `<circle cx="??" cy="??" r="?">`
<svg height="100" width="100" className="bg-gray-300 mx-auto">
  <circle r="50" cx="50" cy="50"></circle>
</svg>
- `<rect x="??" y="??" width="??" height="??">`
<svg height="100" width="100" className="bg-gray-300 mx-auto">
  <rect x="10" y="10" width="20" height="20"></rect>
  <rect x="35" y="10" width="20" height="35"></rect>
  <rect x="65" y="10" width="20" height="50"></rect>
  <rect x="10" y="90" width="80" height="5"></rect>
</svg>

- `<g>` is used to group elements. This helps to apply styles and transformation to all child elements.
<svg height="100" width="200" className="bg-gray-300 mx-auto">
  <circle r="50" cx="50" cy="50"></circle>
  <g transform="translate(100,0)">
    <rect x="10" y="10" width="20" height="20"></rect>
    <rect x="35" y="10" width="20" height="35"></rect>
    <rect x="65" y="10" width="20" height="50"></rect>
    <rect x="10" y="90" width="80" height="5"></rect>
  </g>
</svg>

- `<path d="??">` describes any polygon. It needs a starting point (`M x,y`) and then a line (`L x2,y2`).
<svg height="100" width="100" className="bg-gray-300 mx-auto">

    <path
      stroke="black"
      strokeWidth="2"
      fill="none"
      d="M 10,5 L 10,95"
    ></path>
    <path
      stroke="black"
      strokeWidth="2"
      fill="none"
      d="M 10,95 L 95,95"
    ></path>
    <line stroke-weight="3" stroke="red" x1="15" y1="85" x2="95" y2="10"> </line>
</svg>
